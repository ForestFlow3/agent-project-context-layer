## English

Source ID: S002

Date: 2026-06-09

Current decision:

Start with a static local web app. The first demo should not include a backend.

Preferred stack:

- Single HTML file or small static app.
- Local browser storage is acceptable for demo data.
- No login.

First demo should show:

3. Selected idea detail.

Concern:

The useful wedge is lightweight idea context, not full content generation.
