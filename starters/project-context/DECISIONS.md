## English

Record durable project decisions here.

| ID | Date | Decision | Reason | Status | Sources |
|---|---|---|---|---|---|
| D001 | YYYY-MM-DD |  |  | proposed |  |

### Rule

Do not rewrite accepted decisions without user approval.

Add a new row if a decision changes.

Record accepted choices and why they were made. Do not record temporary ideas that are still being discussed.

If the source is unclear, leave it blank and add an open question in `PROJECT_CONTEXT.md`.
