# RISKS

## 中文

在这里记录已知风险和缓解方式。

| ID | 风险 | 严重性 | 缓解方式 | 来源 |
|---|---|---|---|---|
| R001 |  | medium |  |  |

### 规则

如果风险会影响范围、隐私、对外发布、可靠性或用户信任，先在这里记录，再行动。

风险变化时，更新缓解方式或新增一行。不要删除历史风险，除非用户明确要求清理。

如果风险已经解决，可以在缓解方式中写明当前状态。

## English

Record known risks and mitigations here.

| ID | Risk | Severity | Mitigation | Sources |
|---|---|---|---|---|
| R001 |  | medium |  |  |

### Rule

If a risk affects scope, privacy, external publishing, reliability, or user trust, record it here before acting.

When a risk changes, update the mitigation or add a new row. Do not delete historical risks unless the user explicitly asks.

If a risk has been resolved, note the current status in the mitigation field.
